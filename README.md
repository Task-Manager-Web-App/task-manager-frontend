# Task Manager Frontend

A React + TypeScript task management application with Supabase authentication.

## Quick Start

### 1. Clone the Repository
```bash
git clone https://github.com/Task-Manager-Web-App/task-manager-frontend.git
cd task-manager-frontend
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Run the Development Server
```bash
npm run dev
```

The app will open at `http://localhost:5173`

## Prerequisites

- Node.js (v16 or higher)
- npm or yarn
- Backend server running on `http://localhost:3000`

## Features

- User authentication with Supabase
- Create, read, update, and delete tasks
- User profile management
- Admin and user roles

## Environment Setup

Make sure your backend API is running on `http://localhost:3000` before starting the frontend.
